Animato is CSS based animation framework, Built with Less.js

GETTING STARTED

Simply add "duration" and "effect" classes in your element. You can add "delay, Pause and play, loop, easing and direction" classes for highly customized animaiton effects.

<div class="duration-1s shake">I'm Animation</div>
